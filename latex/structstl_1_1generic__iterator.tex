\subsubsection*{template$<$class T, bool linked\+\_\+list = false$>$\newline
struct stl\+::generic\+\_\+iterator$<$ T, linked\+\_\+list $>$}

Generic iterator container. 

meminc controls the iteration methof If it is set to true, the inc/dec operators will increment memory addresses

Otherwise, this will assume that T is a doubly linked list and use the next and prev functions that MUST be provided by T.
