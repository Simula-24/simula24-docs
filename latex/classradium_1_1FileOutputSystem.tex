\doxysubsection{Detailed Description}
Write logs to a file. 

This will write logs to a file every {\itshape n} logs sent to it This works by using an internal cache of recent logs (\mbox{\hyperlink{classradium_1_1FileOutputSystem_a3cbfc3a6a44176a6bd5954f2d34d35d9}{m\+\_\+log\+Cache}})

You can configure this in the constructor by setting the {\ttfamily flush\+Every\+NLogs} parameter

Any remaining logs will automatically be flushed during destruction
