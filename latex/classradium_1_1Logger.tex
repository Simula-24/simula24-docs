\doxysubsection{Detailed Description}
Logging Class. 

The \mbox{\hyperlink{classradium_1_1Logger}{Logger}} class is responsible for constructing output

Output of the logs is delegated to an \mbox{\hyperlink{classradium_1_1OutputSystem}{Output\+System}}

\mbox{\hyperlink{classradium_1_1Logger_a63fd03daca04addca8bff12d46f15850}{Logger\+::set\+Output\+System$<$\+Type$>$()}} creates a new output system

Logger\+::set\+Output\+System$<$\+Type$>$(\+Construction\+Args\&\&...) creates a new output system and passes args... to its constructor
