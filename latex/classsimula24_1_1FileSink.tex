\doxysubsection{Detailed Description}
Write logs to a file. 

This will write logs to a file every {\itshape n} logs sent to it This works by using an internal cache of recent logs (m\+\_\+log\+Cache)

You can configure this in the constructor by setting the {\ttfamily flush\+Rate} parameter

Any remaining logs will automatically be flushed during destruction
