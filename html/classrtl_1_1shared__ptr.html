Detailed Description

template<class T>
class rtl::shared_ptr< T >

Generic Shared Pointer class.

create via rtl::make_shared <YourType>(YourArgs...);

This is a reference counted Smart Pointer. It will use a structure called __ref_cnt_ptr to hold the pointer and it's reference count

Copying this class will simply increase the reference count Moving this class will NOT

During destruction, this class will try to release itself

  1. if the reference count is 1, we are the owner: destroy the data
  2. otherwise, decrement the refcount
