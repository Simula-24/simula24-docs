Detailed Description

Basic Thread class.

This relies on the operating system's thread API Specifically Window's Thread API and pthreads on *nix Implementations are found in /platform/<name>/thread/thread.cpp

To use:

radium::thread mythread(myfunction, myargs...);
mythread.join();
