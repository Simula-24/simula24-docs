Detailed Description

Print logs to an output stream such as stdout.

By default, all logs will be sent to stderr however, this can be configured via the constructor

You can pass a FILE pointer to a file you created, e.g. 'log.txt', however this is not recommended, as formatting with color may screw with your output Instead, it is recommended to use FileOutoutSystem
