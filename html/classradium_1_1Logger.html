Detailed Description

Logging Class.

The Logger class is responsible for constructing output

Output of the logs is delegated to an OutputSystem

Logger::setOutputSystem<Type>() creates a new output system

Logger::setOutputSystem<Type>(ConstructionArgs&&...) creates a new output system and passes args... to its constructor

Logger::info, Logger::debug, etc. are wrappers around Logger::log
